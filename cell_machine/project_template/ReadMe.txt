========================================================================
КОНСОЛЬНОЕ ПРИЛОЖЕНИЕ. Общие сведения project_template
========================================================================

Это приложение project_template создано автоматически с помощью 
мастера приложений.

   В этом файле представлена сводка содержимого всех файлов, входящих в 
   состав приложения project_template.


project_template.vcproj
   Основной файл проекта VC++, автоматически создаваемый с помощью 
   мастера приложений.
   В файле представлены сведения о версии Visual C++, использованной 
   при создании файла, а также о параметрах платформы, конфигурации и 
   проекта, заданных с помощью мастера приложений.

project_template.cpp
   Основной исходный файл приложения.

/////////////////////////////////////////////////////////////////////////////
Другие стандартные файлы:

StdAfx.h, StdAfx.cpp
   Эти файлы используются для построения файла предкомпилируемых заголовков 
   (PCH) project_template.pch и файла предкомпилируемых типов 
   StdAfx.obj.

Общие заметки:

С помощью комментариев «TODO:» в мастере приложений обозначаются фрагменты 
исходного кода, которые необходимо дополнить или изменить.
